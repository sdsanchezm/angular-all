## misc

- install cli `npm install -g @angular/cli`
- initialize a projects (2 ways): 
    - `npm init @angular myApp`
    - `ng new my-app`
- help
    - `ng --help`
    - `ng generate --help`
    - `ng build --help`
- serve
    - `ng serve`
- angular file:
    - `angular.json`
- tree-shaking
    - is a concept that means: remove all not required when building
